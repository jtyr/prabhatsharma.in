Benefits of microservices

  1. Isolation
  2. Productivity
  3. Faster project development
  4. Evolutionary
  5. Polyglot

Challenges with microservices

  1. Increase in complexity of architecture
  2. Testing & debugging becomes difficult
  3. Steep learning curve of distributed architectures

How to solve these challenges?

  1. Containers and container orchestrators
  2. Serverless compute
  3. Service mesh
  4. Observability tools
